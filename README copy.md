## 安装
建议使用pnpm 安装 node 版本18以上

### 安装依赖
```shell
pnpm install
```

### 启动开发服务
```shell
pnpm run dev
```

### 更新api
```shell
pnpm run api2ts-bot
```

### 发布测试环境
```shell
pnpm run deploy
```
